# docker_env

This is a project that contains multiple `dockerfiles` for multiple dependencies across projects. 

## Current Docker environments

`c_env`: Environment for C++ testing using Google Test Framework for testing and CMake as main building tool.
It contains other tools for better coding practices and documentation such as cpplint and Doxygen.
`ml-env`: Python Environment for using ML models through scikit-learn and Pytorch frameworks.
